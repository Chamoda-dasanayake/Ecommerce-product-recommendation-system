# E-Commerce Product Recommendation System

## Project Overview
This project is a **full-stack E-Commerce Product Recommendation System**. It provides personalized product suggestions using **Collaborative Filtering (CF)** and is designed to be extended with **Content-Based Filtering (CBF)**. Users can discover products they might like based on their interactions and preferences.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| Collaborative Filtering | Recommends products based on similar users' purchase or rating history |
| Content-Based Filtering | Recommends products similar to a selected product (planned) |
| Hybrid Recommendation | Combines CF + CBF for better accuracy (planned) |
| Top Products | Displays most popular products across the platform |

---

## 🏗️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React.js |
| Backend | Python + Flask |
